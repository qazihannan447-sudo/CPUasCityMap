# CPU City

An interactive computer architecture visualizer where the CPU components are represented as a functional bird's-eye city map.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Components**: Radix UI / Shadcn
- **AI**: Google GenAI (Genkit)
- **Icons**: Lucide React
- **Fonts**: Inter (UI), JetBrains Mono (Code)

## Design Palette
- **Primary**: #1D9E75 (Educational Teal)
- **Registers**: #185FA5 (Apartment Blue)
- **ALU**: #0F6E56 (Factory Green)
- **Memory**: #BA7517 (Warehouse Amber)
