# **App Name**: CPU City

## Core Features:

- Live City IDE: An interactive code editor with syntax highlighting for assembly-like instructions, updating city logic in real-time.
- AI Instruction Architect: An AI-powered tool to help students generate or debug complex assembly snippets (like sorting or recursion) for immediate city visualization.
- Metropolitan Logic Grid: A high-fidelity city map representation where Registers, ALU, and Memory function as functional districts and buildings.
- Data Bus Transit System: Dynamic SVG-based road network with animated vehicles representing data packets traveling between CPU components.
- Visual Execution Log: A detailed step-by-step history trace with chronological markers, letting users follow the 'evolution' of the city's state.
- District Status Dashboards: Individual inspection cards for 'Registers', 'Stack', and 'Memory' sectors with distinct accent branding for at-a-glance monitoring.

## Style Guidelines:

- The palette is anchored in 'Educational Clarity' and 'Architectural Warmth.' A primary teal green (#1D9E75) serves as the primary functional accent, set against a warm, refined cream sand background (#FAF7F2).
- Functional zoning colors include architectural blue (#185FA5) for registers, warehouse amber (#BA7517) for memory, garage rose (#993556) for the stack, and dark teal factory green (#0F6E56) for the ALU.
- Font recommendation: 'Inter' (sans-serif) for all UI labels and section headers, and 'JetBrains Mono' (monospace) for register values, memory cells, and code IDE. Note: currently only Google Fonts are supported.
- Minimalist isometric building icons represent system hardware: factory gears for calculation, stacked layers for memory, and small apartments for storage units.
- A strictly organized 3-column dashboard: fixed control sidebar (280px), flexible central viewport map, and data-density state inspector (260px).
- Subtle 150ms ease transitions on state changes and SVG-path vehicle animations simulating 10px wide road transit for data flow.
